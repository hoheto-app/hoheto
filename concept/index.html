Origin
なぜ今、これが必要か

2001年、日本にひとつのサービスが生まれた。ブログもSNSも存在しなかった時代に、「関心でつながる」という発想を実装した関心空間だ。グッドデザイン賞を受賞し、約6万人のユーザーが集った。しかし2016年、「情報と情報がつながる場としての役目を終えた」という言葉とともに閉鎖された。

その後、SNSは爆発的に普及した。しかしFacebookは人間関係の維持装置となり、Xは速度と炎上の場になり、Instagramはビジュアル競争の場になった。どれも「人が先、関心は後」という構造だ。

関心空間が問いかけていたことは、まだ誰も完全には答えていない。Hohetoはその問いへの、現代的な答えだ。

「自分のために書いていたら、
同じ場所に立っている人がいた。」

— Hoheto が実現したい体験

Cultural Root
いろはに ほへと
いろは歌 — 日本語の源流
Hoheto
創業者の名（Horie Hayato）の頭音が、いろは歌の「ほへと」と重なる。
個人の必然と文化的な文脈が、一つの名前に収束した。

第一の必然 — いろはの続き。「いろはにほへと」は平安時代から続く、すべての仮名文字を網羅した歌だ。「いろは」が始まりなら、「ほへと」はその次へ向かう言葉。関心空間の精神を受け継ぎ、その先へ進むというコンセプトと完璧に一致する。

第二の必然 — 創業者の名前。Horie Hayatoの頭音Ho・H・oが「ほへと」の三音と重なる。名前がサービス名に宿るという、語り継がれるブランドストーリーが生まれた。

第三の必然 — 意味の空白。「ほへと」は意味を持たない音の連なりだ。だからこそ、Hohetoというサービスが新しい意味を吹き込むことができる。名前が白紙であることが、最大の強みになる。

Core Concept
関心が先、人は後から

Hohetoは「書くこと」を中心に置く。投稿ではなく、育てること。誰かに見せるために書くのではなく、自分のために深く書いていたら、静かに同じ関心を持つ人が現れる—そういう場所だ。

書く体験
書いている間は完全に一人。邪魔されない、静かなアーカイブ。noteのように長く深く書ける。
出会いの構造
アルゴリズムなし。自分でつけたタグが他者のタグと重なったときだけ、接点が生まれる。
反応の設計
いいね数もフォロワー数もない。「この関心を私も持っています」という共鳴だけ。数は非表示。
つながりの単位
人をフォローするのではなく、キーワードに引き寄せられる。「この人が好き」でなく「この関心が好き」。

Mechanics
五つの設計原則
  • ほ
    キーワードを育てる
    投稿の単位は「キーワード」。モノ、場所、概念、記憶—何でもよい。一度書いて終わりではなく、時間をかけて加筆・深化させる生きたドキュメント。
  • へ
    熟成してから回廊へ
    書いてすぐ公開ではなく、自分だけが読み返す期間を経て、満足したときに初めて他者の目に触れる。衝動的な発信を防ぐ構造。
  • と
    タグだけが道をつなぐ
    アルゴリズムによるレコメンドなし。自分でつけたタグが他者のタグと重なったときだけ、静かに通知が届く。偶然性は人工的に演出しない。
  • ち
    数字を消す
    フォロワー数、いいね数、閲覧数—すべて非表示。唯一の反応は「共鳴」ボタンのみ。書く動機が承認欲求ではなく純粋な関心であるための設計。
  • り
    関心の時間軸を持つ
    関心は変化する。「以前はこう思っていたが今はこう」という変容が記録され、自分も他者もその軌跡を辿れる。静的なプロフィールではなく、動的な知的自伝。

Roadmap
三つのフェーズ
Phase 1 — 種を蒔く
個人アーカイブとしての完成
まず「極私的関心空間」として自分一人で使える道具を完成させる。キーワード登録・タグ・熱量・タイムライン・AI分析。ユーザー自身が最初の証人になる。
Phase 2 — 回廊を開く
静かなコミュニティの形成
招待制で少数のユーザーに開放。タグによる偶然の出会い、共鳴機能、キーワード単位の対話を実装。数字ではなく質でつながるコミュニティを育てる。
Phase 3 — 宇宙へ
国際展開・会社化
Hoheto Inc.として法人化。多言語対応、APIの公開、持続可能なビジネスモデル（広告なし・サブスクリプション）の確立。関心の地図を世界規模で描く。
Vision
私たちが作りたい世界

インターネットはかつて、偶然の発見に満ちていた。検索していないものに出会い、知らなかった人の言葉に心を動かされた。アルゴリズムはその偶然性を、予測可能な快楽に変えてしまった。

Hohetoは、その偶然性を人間の手に戻す試みだ。自分の関心を丁寧に言語化し、静かに世界に置いておく。そこに同じ場所に立っている誰かが、ある日静かに現れる。

古書店で、見知らぬ人が同じ本を手にしている瞬間のような出会い。それがHohetoの目指す体験だ。

いろはにほへと　ちりぬるを—
散っても、また集まる。
すべての関心が、ここへと向かう。

— Hoheto tagline
